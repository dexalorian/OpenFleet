# OpenFleet

Open Cross-platform Saas for operating taxi and cargo fleet.
Fullstack app, node/vue.

<p>TypeScript</p> 
<p>Vue (SFC)</p>
<p>Livekit TURN for video/audio</p>
<p>Websockets</p>

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
